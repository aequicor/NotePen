## Почему

Экран просмотра PDF (`DetailsContent`) не имеет явной кнопки возврата к списку
файлов. Пользователь вынужден использовать системный жест «назад» или
аппаратную кнопку, что неочевидно, особенно на планшетах и десктопных
платформах. Круглая кнопка с иконкой стрелки в левом верхнем углу делает
навигацию доступной и согласованной с Material Design 3.

## Критерии приёмки

| ID | Дано | Когда | Тогда |
|----|------|-------|-------|
| AC-1 | Пользователь открыл экран просмотра PDF | Экран отображается | В левом верхнем углу видна круглая кнопка с иконкой стрелки «←» |
| AC-2 | Кнопка «Назад» отображается | Пользователь нажимает кнопку | Вызывается `component.onBack()` и пользователь возвращается на предыдущий экран |
| AC-3 | Экран просмотра PDF загружается | Любой момент отображения | Кнопка «Назад» находится поверх контента PDF (не перекрывается `LazyColumn`) |
| AC-4 | Экран просмотра PDF отображается | Пользователь видит кнопку | Кнопка имеет круглую форму и соответствует Material3 |
| AC-5 | Экран просмотра PDF отображается | Кнопки «Назад» и масштаба видны одновременно | Кнопка «Назад» не перекрывает кнопки масштаба (`+` / `-`) в правом нижнем углу |

## Граничные случаи

| ID | Серьёзность | Сценарий | Ожидаемое поведение |
|----|-------------|----------|---------------------|
| EC-1 | High | Пользователь быстро нажимает кнопку «Назад» несколько раз подряд | Только первое нажатие обрабатывается (защита от двойного pop стека); последующие нажатия до завершения навигации игнорируются или безвредны (Decompose `pop` идемпотентен при пустом стеке) |
| EC-2 | High | Кнопка «Назад» нажата в момент, когда PDF-страница ещё рендерится | `component.onBack()` вызывается немедленно; рендеринг прерывается через `DisposableEffect(pdfManager) { onDispose { pdfManager.close() } }` — утечки ресурсов нет |
| EC-3 | Medium | Устройство с очень маленьким экраном (ширина < 320 dp) | Кнопка не выходит за границы экрана; размер кнопки фиксирован в соответствии с токенами дизайн-системы |
| EC-4 | Medium | Кнопка «Назад» и область прокрутки `LazyColumn` перекрываются по z-order | Кнопка рендерится поверх `LazyColumn` (помещена после него в том же `Box`); касание кнопки не передаётся в `LazyColumn` |
| EC-5 | Low | Тёмная тема оформления | Кнопка использует цвета из Material3 `FloatingActionButton` / `IconButton` — автоматически адаптируется к теме |

## Как это работает

### Позиционирование

`DetailsContent` уже использует `Box(modifier.fillMaxSize())` как корневой контейнер.
Кнопка «Назад» добавляется как дочерний элемент этого `Box` с выравниванием
`Alignment.TopStart`, после объявления `LazyColumn` (выше по z-order).

### Компонент кнопки

Используется `SmallFloatingActionButton` из Material3 (или `FloatingActionButton`
с размером `FabSize.Small`). Иконка: `Icons.AutoMirrored.Filled.ArrowBack`.
Нажатие: `onClick = { component.onBack() }`.

Альтернатива — `IconButton` с `Modifier.clip(CircleShape)` и явным фоном.
Выбор между ними определяется при реализации на основе доступных токенов
Material3 и соответствия дизайн-системе.

### Публичные сигнатуры (без изменений)

```
// DetailsComponent.kt — без изменений
fun onBack(): Unit

// DetailsContent.kt — изменяется только тело composable
@Composable
fun DetailsContent(component: DetailsComponent, modifier: Modifier = Modifier)
```

Изменений в `DetailsComponent`, `DefaultDetailsComponent`, `DefaultRootComponent`
не требуется — `onBack()` уже реализован и вызывает `navigation::pop`.

### Отступы и размеры

Кнопка позиционируется с `padding(16.dp)` от края экрана. Размер — стандартный
Small FAB (40 dp). Значения должны соответствовать токенам дизайн-системы проекта,
если таковые определены; иначе использовать Material3-дефолты.

### Обработка повторных нажатий

Decompose `StackNavigation.pop()` безопасен при пустом стеке (не бросает
исключение). Дополнительная блокировка на уровне UI не требуется.

## UI / UX

Круглая кнопка («Small FAB» или `IconButton` с `CircleShape`), расположена в левом
верхнем углу `DetailsContent`. Отступ от краёв: 16 dp. Иконка:
`Icons.AutoMirrored.Filled.ArrowBack`. Фон и цвет иконки — из Material3-темы
(автоматически светлая/тёмная тема). Кнопка всегда видна поверх контента PDF.
Кнопки масштаба (`+` / `-`) остаются в правом нижнем углу и не конфликтуют с кнопкой «Назад».
